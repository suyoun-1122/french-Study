Yeonjae French V4.4.0 — SVG 캐릭터 적용본

- 쁘띠냥, 치즈냥, 라벤더냥을 각각 독립 SVG 파일로 적용
- 홈 화면용 세 캐릭터 합성 SVG 추가
- SVG 내부에 둥실 움직임, 손 흔들기, 반짝임 애니메이션 포함
- 홈, 학습, 퀴즈, 요리 팝업의 기존 PNG 참조를 SVG로 교체
- 서비스워커 캐시 버전 4.4.0 적용

참고: 이번 SVG는 승인한 캐릭터 일러스트를 SVG 컨테이너에 포함하고, SVG 애니메이션 레이어를 더한 방식입니다. UI에서는 SVG 파일로 동작하며 확대/배치/애니메이션 적용이 가능합니다.
