Yeonjae French Pâtisserie V2.1 - Build 2

적용 방법
1. ZIP의 내용 전체를 GitHub 저장소 최상위 폴더에 업로드합니다.
2. 같은 이름의 index.html, style.css, app.js는 Replace/덮어쓰기를 선택합니다.
3. 기존 data, assets, audio.js, review.js는 그대로 포함되어 있습니다.
4. GitHub Pages 반영에는 보통 1~3분 정도 걸릴 수 있습니다.
5. 화면이 이전 모습이면 브라우저에서 Ctrl+F5로 새로고침합니다.
