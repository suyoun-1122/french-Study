Yeonjae French V3.4.1 수정사항

1. 퀴즈 보기에서 정답이 사라지는 오류 수정
- 기존에는 보기 전체를 섞은 뒤 앞의 4개만 잘라 정답이 제외될 수 있었습니다.
- 이제 정답 1개를 반드시 포함하고, 중복 없는 오답 최대 3개만 추가합니다.

2. 데이터 검증
- data/words.json 총 100개 항목 확인
- ID 중복 없음
- 명사 62개: 관사와 성 정보 일치 확인
- 동사 18개: 활용 정보 필수 항목 확인
- 형용사 20개: 여성형 및 복수형 정보 확인
- 뜻, 예문, 예문 번역의 공란 여부 확인
- `des yeux` 항목을 학습·퀴즈에 적합한 단수형 `un œil` / 복수형 `des yeux`로 수정

3. 캐시 버전 갱신
- 서비스 워커 및 데이터 쿼리 버전을 3.4.1로 변경했습니다.
- GitHub 업로드 후 이전 화면이 남으면 브라우저 새로고침 또는 사이트 데이터 삭제 후 다시 실행하세요.
