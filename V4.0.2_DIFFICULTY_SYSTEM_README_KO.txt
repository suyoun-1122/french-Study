Yeonjae French V4.0.2 - 난이도 3단계 및 오답 반복 학습

변경 내용
1. 단어/학습유형별 Easy, Medium, Hard 난이도를 개별 저장합니다.
2. 같은 단어와 유형을 3회 연속 맞히면 다음 난이도로 올라갑니다.
3. 틀리면 2~3문제 뒤 같은 유형으로 자동 재출제됩니다.
4. 한 문제는 최대 2회까지 반복되어 무한 반복을 막았습니다.
5. 틀린 경우 해당 유형 난이도가 한 단계 내려가며 기초를 다시 확인합니다.
6. 재출제 문제는 오늘의 10문제 완료 수에 중복 계산하지 않습니다.
7. 문제 상단에 현재 Easy / Medium / Hard 단계가 표시됩니다.
8. 정답을 직접 암시하는 문제 이모지는 숨겼습니다.

난이도 기준
- Easy: 같은 품사의 넓은 범위 오답
- Medium: 같은 주제, 같은 단원 중심 오답
- Hard: 같은 주제와 품사에서 가장 가까운 오답 우선

기존 별, 재료, 음식, 도감, 학습 기록은 유지됩니다.
